> Shell 学习过程中的一些记录

### 命令替换
```shell
echo date # date
# `command` 使用单引号将变量包起来
echo `date +%Y` # 2021
# $(command)
echo $(date)   # 2021年 4月21日 星期三 00时44分02秒 CST
```

### 变量替换
```shell
var_1=3
echo ${var_1}
# 计算变量长度
# 第一种方式
echo ${#var_1} # 25
# 第二种方式
len=`expr length "$var_1"`
echo $len # 25
```

### 数学运算
使用双括号进行整数运算

```shell
# (( )) 使用双括号将数学运算包起来
echo $((20+30))
# 50
```
