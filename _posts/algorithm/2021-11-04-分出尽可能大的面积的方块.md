> 总体上还是分而治之的思想

欧几里得算法：两个整数的最大公约数等于其中较小的那个数和两数相除余数的最大公约数

```py
def soilsplit(length, width):
    while length >= width:
        if length%width == 0:
            return width
        else:
            length, width = width, length%width
```
