## JS 中的数据类型

### 原始数据类型

- Boolean ：布尔类型，只有 true 和 false两个值
- undefined ：未定义
- null
- Number：数字类型
- String ：字符串
- Symbol：ECMAScript6新增类型，表示独一无二的值
- Biglnt：ECMAScript2020 新增数据类型
